Tableau Dashboard Collection

Re-Imagine 911 Data Story

This Tableau story (comprised of 7 dashboards) was created for Code for America's Re-imagine 911 project. In the story, we look into the project's context, effectiveness of project data standardization efforts, and analysis that can be performed using data to uncover insights to divert 911 calls to non-police responders and improve the current 911 system.

Re-imagine 911: Open Data Information Lookup System

Created using Tableau to make it easier for people to look up information about a given dataset that was included in the "Open Data Review" phase of the Re-imagine 911 project.

Analyzing Worldwide COVID Impact and Vaccination

Inspired by YouTuber AlextheAnalyst, I followed him in creating a basic dashboard using public COVID-related data and going one step further to create an interactive Tableau story that also serves as an exploration tool to look into COVID impact on different groups of people and compare vaccination efforts and infection rate.

General Electric Manufacturing Quality

Created for Forage's General Electric Virtual Internship.

As per task requirements, I created a dashboard story in Tableau containing insights and metrics that may be of interest to General Electric. Using data provided, I assessed error rates across General Electric's manufacturing operations by looking at whether aviation features/components manufactured meet measurement requirements.

Comic Book Industry and Cultural Influences

Earning 1st place in class, my team and I dug into publicly available datasets on Marvel and DC superheroes to see if we can uncover any interesting trends and patterns. Surprisingly, we discovered how comicbook superhero creation trends are a great reflection of societal views and political events happening then.
